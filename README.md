AI Interview Preparation Assistant

A Python-based desktop application that helps users prepare for technical interviews through role-based mock interview sessions. The application provides interview questions, evaluates responses with a basic scoring system, and stores interview results using SQLite.

Features

* Candidate name input
* Role-based interview questions
* Interactive GUI built with Tkinter
* Automatic interview scoring
* SQLite database integration for storing results
* Simple and beginner-friendly project structure

 Technologies Used

* Python 3
* Tkinter
* SQLite
* Git
* GitHub
* Visual Studio Code



 Project Structure


AI-Interview-Preparation-Assistant/

├── feedback.py         # Feedback logic
├── view_results.py     # View saved interview results

 How to Run

1. Clone the repository

```bash
git clone https://github.com/sireesha18022005/AI-Interview-Preparation-Assistant.git
```

2. Open the project folder

```bash
cd AI-Interview-Preparation-Assistant
```

3. Run the application

```bash
py gui.py
```
